# x-element
A Webapp framework designed for Simplicity. Lightweight, fast and standards-compliant. 

Features:
- client-side HTML preprocessing via xslt.

Development:
- Browser: open index.xhtml
- Build: `xsltproc index.xhtml > index.html`
